# GeeksforGeeks-Crawler

Crawler based on GeeksforGeeks tags

## Installation

*Requirements* : Python 3.6, pipenv (`pip install pipenv`)
In the project folder `pipenv sync` and you're done. 


## Usage
`scrapy crawl -o data.json -a tag=amazon`
